# Roman Personal Training — Gym Timer App

Aplicatie personala de antrenament cu timer de pauza, rapoarte zilnice, calendar si grafic de progres.

## Functii
- ⏱ Timer de pauza cu alarma si vibratii
- 📋 Rapoarte zilnice de antrenament
- 📅 Calendar cu planificare
- 📊 Grafic de progres (saptamanal / lunar)
- 🎤 Recunoastere vocala (Chrome Android)
- 📱 Instalabila ca app pe telefon (PWA)

## Instalare pe telefon
1. Deschide link-ul GitHub Pages in **Chrome**
2. Apasa meniul ⋮ → **Add to Home Screen**
3. Apasa **Add** — apare pe ecran ca o aplicatie reala

## GitHub Pages
Settings → Pages → Branch: main → Save
